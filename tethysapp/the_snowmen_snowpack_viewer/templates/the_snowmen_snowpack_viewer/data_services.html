    Data Services

    The Snow Pack Project uses:
  • The Snotel Network Stations in the State of Utah
  • The Counties in the State of Utah
  • The Digital Elevation Model

    • The shapefile of Snotel Network Stations is constructed from the database of SNOTEL stations and it is hosting as a Feature Service in the BYU Geoserver. These stations provide snowfall data that helps the application calculate the changes in snowpack over time and is provided using a feature service. The raw data can be found here: https://wcc.sc.egov.usda.gov/nwcc/rgrpt?report=daily_snowdelta_7day&state=ut

    The shapefile of Utah Counties is downloaded from the Utah Automated Geographic Reference Center and it is hosting as a Feature Service in the BYU Geoserver.

    The Utah Digital Elevation Model is a rastter file downloaded from the Hydrological data and maps based on SHuttle Elevation Derivatives at multiple Scales that is based on high-resolution elevation data obtained during a Space Shuttle flight for NASA's Shuttle Radar Topography Mission (SRTM) and it is hosting as a Map Service in the BYU Geoserver.
