    Instructions

    This Tool allows the user to perform a regional analysis of snow depth data for a watershed of interest. This is meant to be applicable for estimating the SWE and ultimately the runnoff in watersheds that feed local reservoirs. The following instructions indicate how to use the tool:

    1.) Click on a stream in the Flow Accumulation Raster displayed in the map window to delineate a watershed.

    2.) Once You have created your outlet point, click on the Delineate Watershed button.

    3.) When the service is finished a polygon will be generated for the watershed.

    4.) Make sure you have the desired date selected.

    5.) Click the Submit Region button. When the service is finished processing, it should display your resulting raster.

    5.) To perform another analysis click the Refresh button.

    The snowtel stations and flow accumulation raster can still be toggeled on an off with the blue checkbox. When the user clicks on a snotel station, a popup is generated with the station information and a time series of all the available snowdepth date for that station is generated below the map.
