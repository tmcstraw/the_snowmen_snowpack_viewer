Data Services talks about the data services used in our Geoprocessing tools, how the data were created, where are the data hosting, and what data is included

The Animation Tool allows the user to view the each of the rasters for January 2017 and view an animation file video file which plays the rasters through the month of January. The Polygon Tool allows the user to draw a polygon around any region in utah(a ski resort for example) and creates a new result file with contours based on the data from that region. The Watershed Tool does this for a watershed that is specified by a user defined point(outfall). Each of these three map pages(Animation, Polygon, and Watershed) also allows the user to view time series data for each of the snotel stations in the state of Utah.

The about page include a little bit about who the authors are and the links to contact them.

Instructions

Select one of the tools in the App Navigation menu on the left side of the page.

The Animation Tool allows the user to view the rasters for the enitre state for any day in January 2017.

The Polygon Tool allows the user to perform a regional analysis for snow depth on a specific area in Utah (a ski resort for example).

The Watershed Tool performs a snow depth analysis for a watershed that calculated from a user defined outlet point. This helps the user get an idea of how much SWE may reach the reservoir during runoff season and other useful calculations.

The About page provides further information about the application, its development, and its authors.
